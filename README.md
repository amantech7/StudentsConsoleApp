# StudentsConsoleApp

A simple Python console application that registers students and shows their details.

## Features
- Collects student details via console prompts
- Shows greeting (Hello FirstName)
- Displays full student form after registration
- Allows adding multiple students
- Can show all registered students
- Shows total student count

## How It Works
1. Application asks student form questions one by one
2. After submission it prints:
   - Hello “First Name”
   - Complete student details
3. Menu provides options:
   - Add More Students
   - Show All Students
   - Show Total Registered Students
   - Exit
